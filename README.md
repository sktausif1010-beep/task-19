# Service Booking Web Application

## Project Overview
This is a simple Service Booking Web Application built using HTML, CSS, and JavaScript. Users can browse available services, add them to their booking list, view the total amount, and complete the booking by filling out a booking form.

## How to Run

1. Download the project.
2. Extract the ZIP file.
3. Open the project folder.
4. Open `index.html` in any modern web browser.
